const int InsideTolerance = 64

The tolerance for determining we are inside a tile

Our tiles are a diamond that is 32 pixels tall and 64 pixels wide. So, if we take the distance from the center vertically and double it, it would be as if we had a 64 by 64 diamond. The "Manhattan distance" from the center would be no more than 64 in that case.
