Detailed Description

Implementation of Canny edge detection. This is the base (reference) implementation.

See http://en.wikipedia.org/wiki/Canny_edge_detector for a complete description of this method.

void CCanny::TraceEdges ( CArray2D< double > &  gradient,
std::list< wxPoint > &  edges 
) [private]

Compute a list of edge pixels. Finding starting points in the gradient image where the gradient is high, this function follows the edge using a lower threshold to trace out the edge continuity. It calls the recursive function Trace that does the actually tracing.

Notice: This process destroys the gradient image.

Parallelizing this function will be much harder than the other functions.

Parameters:
gradientThe gradient image after non-maximum suppression
edgesThe output edge pixels that have been traced.
